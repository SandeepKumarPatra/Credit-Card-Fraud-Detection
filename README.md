# Credit Card Fraud Detection: Handling Imbalanced Data using Under-Sampling and Over-Sampling

This project demonstrates how to handle severe class imbalance in the Credit Card Fraud Detection dataset using resampling techniques from Scikit-learn.

## Techniques Used
- Under Sampling (Reducing Majority Class)
- Over Sampling (Duplicating Minority Class)
- Pandas & NumPy for data processing
- sklearn.utils.resample for dataset balancing

## Objective
Fraud detection datasets are highly imbalanced. This project balances the dataset to improve machine learning model performance.
